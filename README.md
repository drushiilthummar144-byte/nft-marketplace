# NFT Marketplace - Setup Guide

## Prerequisites
- Node.js (Latest LTS version recommended)

## How to Run on a New PC
1.  **Unzip** the folder `nft-marketplace-package.zip`.
2.  Open a terminal (Command Prompt or PowerShell) inside the folder.
3.  Install dependencies:
    ```bash
    npm install
    ```
4.  Start the server:
    ```bash
    node server.js
    ```
5.  Open your browser and visit: `http://localhost:3000`

## Features
- **Public**: Browse NFTs, Search, Contact Form.
- **User Profile**: Edit Profile, Manage Created/Collected items, Change Password.
- **Admin Panel**: Dashboard Stats, User Management (Ban/Role), CMS (Edit Site Content), Settings.
